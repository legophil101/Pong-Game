# Pong Game in Python

A simple **Pong game** built in Python using the `turtle` module, featuring **OOP structure**, **continuous paddle movement**, **collision handling**, and **sound effects** with `pygame`.

## Features
- Two-player local multiplayer
- Continuous paddle movement
- Ball collision with walls and paddles
- Ball speed increases after paddle hits
- Sound effects for paddle bounce and scoring
- Scoreboard display
